# .NET Coding Interview Preparation

Repository for .NET coding interview preparation.
This repository contains various code samples, algorithms, data structures, and test cases to help you prepare for .NET developer interviews.

## Directory Structure

The repository is organized as follows:
```shell
├── Algo
│ └── ...
├── Puzzle
│ └── ...
├── Readme
│ └── ...
├── UnitTest
│ └── ...
└── README.md
```
- **Algo**: Contains various algorithm implementations such as sorting, searching, etc.
- **Puzzle**: Contains solution to common coding puzzles etc.
- **Readme**: Contains text, documentations and explanation of some Algo.
- **UnitTest**: Contains some test cases of Algo.
